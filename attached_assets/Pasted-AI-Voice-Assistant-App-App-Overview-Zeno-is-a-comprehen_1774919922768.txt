AI Voice Assistant App
📱 App Overview
Zeno is a comprehensive voice-controlled personal assistant for Android built with React Native/Expo. It combines AI conversation capabilities with deep Android system integration for hands-free phone control.

🎯 Core Features
1. AI Conversation
Powered by LLaMA 3.3 via OpenRouter API
Natural language understanding and context retention
Voice input with audio recording
Text-to-speech responses for hands-free operation
100+ voice command variations recognized
2. Phone Controls
Flashlight: Toggle on/off via voice
Volume: Adjust media, ringer, alarm volumes
Screen Brightness: Control display brightness levels
Device Lock: Lock phone instantly
WiFi: Enable/disable WiFi
Bluetooth: Toggle Bluetooth connectivity
3. Message Reading & Sending
Read WhatsApp and SMS messages via AccessibilityService
Auto-capture new messages in real-time
Display messages in dedicated Messages tab
Voice-controlled message sending (planned)
4. Web Search Integration
DuckDuckGo API for web searches
Voice-activated search queries
Display search results with titles, snippets, and links
Inline result cards in chat interface
5. Background Service
Android Foreground Service for persistent execution
Keeps assistant responsive even when app is minimized
Persistent notification for service status

🔧 Key Services
1. openRouterService.ts
Handles LLaMA API requests
Streams responses for real-time conversation
Manages conversation context and history
2. commandParserService.ts
Recognizes 100+ voice command variations
Routes to appropriate phone control or search
Examples: "turn on flashlight", "set volume to 50%", "search for restaurants"
3. phoneControlService.ts
Bridge to native Android PhoneControlModule
Executes system-level actions
Returns success/error feedback
4. messageService.ts
Bridge to AccessibilityService
Fetches captured WhatsApp/SMS messages
Real-time message polling
5. voiceService.ts
Audio recording using Expo Audio
Returns audio URI for transcription
Handles permissions and errors
6. ttsService.ts
Text-to-speech playback
Converts assistant responses to audio
Cross-platform Audio API
7. searchService.ts
DuckDuckGo API integration
Parses search results (title, snippet, URL)
Returns formatted search cards
8. backgroundService.ts
Starts/stops Android Foreground Service
Persistent notification management

Special Permissions (via Settings)

BIND_DEVICE_ADMIN - Device lock functionality
Normal Permissions (auto-granted)
INTERNET - API calls
ACCESS_WIFI_STATE / CHANGE_WIFI_STATE - WiFi control
BLUETOOTH / BLUETOOTH_ADMIN - Bluetooth control
MODIFY_AUDIO_SETTINGS - Volume control
WAKE_LOCK - Keep service alive
FOREGROUND_SERVICE - Background execution

💡 Usage Example
Voice Command Flow:

User taps microphone button in Chat tab
Speaks: "Turn on the flashlight"
App records audio and processes command
Command parser identifies flashlight control
Calls phoneControlService.toggleFlashlight()
Native module enables flashlight
Assistant responds: "Flashlight turned on"
TTS plays audio confirmation
Message Reading Flow:

WhatsApp notification arrives
AccessibilityService intercepts notification
Captures sender, message text, timestamp
Stores in service memory
Messages tab auto-refreshes
Displays new message in list
